# broken-branch-strategy-lab

This repo simulates unstable Git collaboration practices for educational recovery exercises.

The goal is for students to repair this repository using proper branching strategy and GitHub Flow practices.

## Current Mess
- Direct commits on main
- Overlapping unfinished feature work
- Inconsistent branch naming
- Stale branches
- Merge conflicts
- Messy commit history
- Unstable integration flow
- No protected workflow discipline
